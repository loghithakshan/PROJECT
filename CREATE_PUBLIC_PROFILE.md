## Step-by-Step: Create New GitHub Repo & Push

### **Step 1: Create New Repository on GitHub.com** (2 minutes)

1. Go to: **https://github.com/new**
2. Fill in the form:
   - **Repository name:** `crowd-assistance`
   - **Description:** "Multilingual AI Emergency Response Platform - 150+ Languages, Real-Time Hazard Detection, Enterprise Security"
   - **Visibility:** Select **PUBLIC** ✓
   - **Initialize repository:** Leave unchecked
   - **License:** MIT (optional)
3. Click **Create repository**

You'll get a new empty repo. Copy the URL shown (looks like: `https://github.com/YOUR-USERNAME/crowd-assistance.git`)

---

### **Step 2: Update Local Git Remote** (1 minute)

In your backend folder, update the git remote:

```powershell
# Navigate to backend folder
cd "c:\Users\rakes\Downloads\in assisto\backend"

# Remove old remote
git remote remove origin

# Add new remote (replace YOUR-USERNAME and repo-name as needed)
git remote add origin https://github.com/YOUR-USERNAME/crowd-assistance.git

# Verify it worked
git remote -v
```

Expected output:
```
origin  https://github.com/YOUR-USERNAME/crowd-assistance.git (fetch)
origin  https://github.com/YOUR-USERNAME/crowd-assistance.git (push)
```

---

### **Step 3: Push Code to New Repository** (2-5 minutes)

```powershell
# Push all code to the new repository
git push -u origin main
```
