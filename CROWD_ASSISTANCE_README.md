## What is CROWD ASSISTANCE?

A production-ready emergency response platform that coordinates responders across language barriers in real-time. When disaster strikes, CROWD ASSISTANCE detects hazards, broadcasts alerts in 150+ languages, and connects emergency teams instantly.

**Live in 5 minutes. Supporting millions. Speaking all languages.**

```
┌─────────────────────────────────────────────┐
│         Frontend / Mobile Apps              │

└────────────────┬────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────┐
│       NestJS Backend (12,000+ LOC)         │
├─────────────────────────────────────────────┤
│ ✓ Auth Module          (500 LOC)            │
│ ✓ Hazard-Oracle        (450 LOC)            │
│ ✓ Alert-Broadcaster    (550 LOC)            │
│ ✓ Translation-Engine   (1,600 LOC)          │
│ ✓ Network-Verifier     (350 LOC)            │
│ ✓ Audit-Ledger         (300 LOC)            │
└────────────┬──────────────────┬─────────────┘
             │                  │
             ▼                  ▼
    ┌───────────────┐   ┌──────────────┐
    │  PostgreSQL   │   │    Redis     │
    │    (Data)     │   │   (Cache)    │
    └───────────────┘   └──────────────┘
```

**Tech Stack:**
- **Language:** TypeScript 5.3+
- **Framework:** NestJS 10.2+
- **Database:** PostgreSQL 16 (Prisma ORM)
- **Cache:** Redis 7
- **Crypto:** Ed25519, XChaCha20-Poly1305, Argon2id, Groth16
- **Containers:** Docker
- **Orchestration:** Kubernetes, Heroku
- **CI/CD:** GitHub Actions

## 🔌 API Endpoints

### Health Check
```bash
GET /health
```

### Translation
```bash
POST /translation/translate
{
  "text": "URGENT: Earthquake detected",
  "sourceLanguage": "en",
  "targetLanguage": "es"
}
```

### Languages
```bash
GET /translation/languages
GET /translation/languages/:code
GET /translation/languages/stats
```

### Hazard Detection
```bash
POST /hazard/detect
{
  "latitude": 37.7749,
  "longitude": -122.4194,
  "reportsData": "earthquake magnitude 5.0"
}
```

### Alerts
```bash
POST /alerts/broadcast
{
  "message": "URGENT: Flood warning",
  "language": "en",
  "recipientGroups": ["emergency_responders"]
}
```
